# Getting Started

### Installation

```bash
yarn add v-burger-menu
```

### Import CSS
```js
import './node_modules/v-burger-menu/dist/v-burger-menu.css'
```

### Import CSS (Nuxt js)
```js
module.exports = {
  ...
  css: [
    ...
    "./node_modules/v-burger-menu/dist/v-burger-menu.css"
  ],
  ...
};

```
